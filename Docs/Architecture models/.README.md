## draw.io images
Some diagram files in this directory are images which can be edited with 
[draw.io](https://www.drawio.com), as marked with the `*.drawio.png` extension.

The drawio images contain a copy of the diagram structure as metadata. This allow the 
diagram to be viewed as an image (makes it handy to view in GitHub or to display 
on websites) while also remaining editable.

[Download and install](https://github.com/jgraph/drawio-desktop/releases) the 
latest release of draw.io to open the image files for editing.

Make edits in draw.io, save the files and commit them to publish them.

## PUML files
* In VS Code, install `jebbs.plantuml` extension 
	* Follow the instructions to **setup a local PlantUML Server as the renderer**
	* Running PlantUML Server with Docker is recommended:
        * Install [Rancher Desktop](https://rancherdesktop.io/) (free to use commercially)
        * `docker run -d -p 8080:8080 --name sui-plantuml plantuml/plantuml-server:jetty`
* In VS Code, open `.puml` file
	* To preview diagram, press `Alt + D` to start PlantUML preview (`option + D` on MacOS).
	* To export, right-click and choose `Export Current Diagram`.
    * Save/move the exported diagrams in to the `Docs/Architecture models/generated` folder.
