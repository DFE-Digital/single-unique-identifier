## Status
Draft — work in progress

---

## Overview

This ADR examines whether national discovery services should be exposed via the
public internet, private networking, or a hybrid model.

It frames the trade-offs between accessibility, onboarding friction, security
posture, and operational complexity, and how these choices affect searchers,
custodians, and administrative users.

This ADR is a placeholder for further analysis and does not yet record a decision.
